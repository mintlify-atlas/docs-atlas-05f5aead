## Description

Get OpenType feature code from the font.

## Parameters

This tool takes no parameters.

## Returns

<ResponseField name="features" type="array">
  Array of OpenType feature objects
  
  <Expandable title="Feature object">
    <ResponseField name="tag" type="string">
      Four-character feature tag (e.g., "liga", "kern", "ss01")
    </ResponseField>
    
    <ResponseField name="name" type="string">
      Human-readable feature name
    </ResponseField>
    
    <ResponseField name="code" type="string">
      OpenType feature code in AFDKO syntax
    </ResponseField>
    
    <ResponseField name="active" type="boolean">
      Whether the feature is enabled
    </ResponseField>
    
    <ResponseField name="automatic" type="boolean">
      Whether the feature is automatically generated by GlyphsApp
    </ResponseField>
  </Expandable>
</ResponseField>

## Usage

```python
import anthropic

client = anthropic.Client()

response = client.messages.create(
    model="claude-3-5-sonnet-20241022",
    max_tokens=4096,
    tools=[
        {
            "type": "mcp",
            "server_name": "glyphs-mcp",
            "tool_name": "get_features",
        }
    ],
    messages=[{"role": "user", "content": "What OpenType features are in this font?"}],
)
```

```bash
# Using Claude Code CLI
get_features()
```

## Example Response

```json
{
  "features": [
    {
      "tag": "liga",
      "name": "Standard Ligatures",
      "code": "feature liga {\n  sub f i by fi;\n  sub f l by fl;\n} liga;",
      "active": true,
      "automatic": false
    },
    {
      "tag": "kern",
      "name": "Kerning",
      "code": "# Automatic Code",
      "active": true,
      "automatic": true
    },
    {
      "tag": "ss01",
      "name": "Stylistic Set 1",
      "code": "feature ss01 {\n  sub a by a.ss01;\n} ss01;",

      "automatic": false
    }
  ]

```

## Notes

- Returns all feature definitions including automatically generated ones (kern, mark, mkmk, etc.)
- Feature code uses AFDKO (Adobe Font Development Kit for OpenType) syntax
- Automatic features are generated by GlyphsApp based on glyph data and cannot be edited
- Custom features can be created or modified using `set_feature_code()`
- Common feature tags:
  - `liga`: Standard ligatures (fi, fl, etc.)
  - `kern`: Kerning
  - `ss01-ss20`: Stylistic sets
  - `salt`: Stylistic alternates
  - `calt`: Contextual alternates
  - `smcp`: Small capitals
